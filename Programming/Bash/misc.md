# Miscellaneous
## Description
Other useful bash commands.

## Random
```bash
top # Display current processes
cal # Display calendar
cal -3 # Display calendar with next and prev month
date # Display date
system_profiler SPDisplaysDataType |grep Resolution # Get system resolution mac
```

## Folder tree
Display a tree for a defined folder

```bash
brew install tree # Install
tree # Run in current dir
tree /folder1/folder2 # Run in defined dir
```

## Newsboat
RSS terminal reader

```bash
brew install newsboat # Install
newboat # Run
vi ~/.newsboat/urls # Add RSS feeds as urls
vi ~/.newsboat/config # Add specific configuration 
```

## File transfer (Android)
Add and remove files from android device on mac. Run from GUI application after installation.

```bash
brew install --cask android-file-transfer # Install
```
 
## Delete folders
```bash
rm -r folder_name # Remove folder and its contents
```
