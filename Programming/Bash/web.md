# Web
Downloading files using bash.

## Curl
```bash
curl http://www.example.com # Show html in terminal
curl http://www.example.com -O example.html # Save html 
curl http://www.example.com/filename.cpp -O "filename.cpp" # Save a file
# or
curl -O http://www.example.com/filename.cpp "filename.cpp"
```

## Wget
```bash
wget http://www.example.com/filename.cpp # Download file
```
